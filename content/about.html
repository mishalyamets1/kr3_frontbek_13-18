О приложении

Версия 1.2.3

Это приложение для заметок разработано как Progressive Web App (PWA) с использованием архитектуры App Shell.

Приложение работает полностью офлайн, хранит заметки в localStorage и использует Service Worker для кэширования ресурсов.

Все статические ресурсы кэшируются при первом посещении, а динамические страницы загружаются по стратегии Network First с fallback на кэш при отсутствии сети.
